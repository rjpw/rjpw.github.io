## Quick Tips

### Run Jekyll and Rebuild Automatically

```sh
jekyll server --watch
```

### Run Jekyll using Bundle

```sh
bundle exec jekyll serve
```

### Include Drafts

```sh
bundle exec jekyll serve --drafts
```
